# QwenPaw

## Что это

QwenPaw — open-source персональный AI-ассистент от AgentScope (Alibaba).

## Параметры

### `qwenpaw_port` (по умолчанию: 8088)
Порт Web UI QwenPaw.

### `auth_enabled` (по умолчанию: false)
Включить авторизацию для доступа к Web UI.

## Доступ

- Web UI: `http://<IP_HA>:8088/`
- Кнопка "Open Web UI" в карточке дополнения

## Данные

Workspace хранится в `/share/qwenpaw/` — доступен по SSH из любого add-on.
